# forecast.py
weather forecast powered by forecast.io

## requirements
- python3
- requests
- prettytable 

## options
```
  <no arg>    shows current weather
  -h, --help  shows help message
  -d          daily forecast (max 8 days)
  -ho         hourly forecast (max 24 hours)
  -l L        limit forecast output to <L> entries
```
